Requirements:

Filza (or any file manager really)

AppSync Unified (cydia.akemi.ai)

Steps:

1 - Download the Unsigned Ipa you want and Open it in Filza

2 - Extract the Ipa in Filza

3 - Go into the folder you extracted into the Payload and copy the .app file

4 - Go to Root section on the side panel and press on the Applications folder you want to paste the .app into that folder

5 - Go to /usr/bin/ and find 'uicache'. Press it, and let it run.

You should now see the app on the home screen. If you have AppSync, you should now be able to open and use it. Enjoy!
